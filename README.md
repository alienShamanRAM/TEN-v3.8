# TEN VFram

This is the new modular migration version of the TEN26 visual app.

You can open `index.html` directly or run it from a tiny local server:

```sh
python3 -m http.server 8097
```

Then open:

```text
http://127.0.0.1:8097/index.html
```

The preserved visual baseline lives in the copied `ten26-*.js` files. The rejected dual-SVG/view-bar experiment is no longer loaded.
