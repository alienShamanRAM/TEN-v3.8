Tooltips

Shortcuts

Up hides the UI. Down restores it. Left and Right perform slide flicker. Space attracts dots to the current slide. Esc stops flicker.

Overview

TEN26 animates active grid layers between grid homes and slide targets on one shared stage. Upload Media supplies vector and raster sources, Timing controls the slideshow motion, Grid Masking shapes how grid-home dots hide and return, and Advanced Options controls layout, flicker style, and blink behavior.

Header Controls

Left arrow or keyboard Left immediately goes to the previous slide. Right arrow or keyboard Right immediately goes to the next slide. The center hold button or Space attracts dots to the current slide while held. Keyboard Up hides the UI, and keyboard Down restores it. The fast-forward button runs automatic slide changes using the current slide source duration.

Grid Layers

Grid Layers controls each active grid layer. Use the eye button to hide a layer, copy layers above or below the selection, rename layers, delete layers, and reorder them. Target chooses whether that layer follows SVG fills, paths, or anchors. The three color groups set Grid, Mid, and Target color, and Mid Point shifts where the mid color and mid size happen between grid and target. The main Random button here only randomizes unlocked values inside the layer stack.

View Options

View Options changes preview zoom, frame cap, fullscreen, and the info overlay. It does not resize the stage or move grid or slide coordinates.

Upload Media

Upload Media groups Vector, Raster, and Grid Masking. Use it to load slide sources, position them on the canvas, and control how grid-home dots react to those sources.

Vector

Vector loads SVG artwork as dot targets. Scale and offset position the SVG on the canvas. Vector slides share the same slide order and transition system as raster media.

Raster

Raster loads PNG, JPG, or video files as rectangular dot targets. Each media file becomes a rectangle with corner anchors, edge paths, and a filled interior. Scale and offset position the media on the canvas.

Grid Masking

Grid Masking holds vector and raster mask controls. Mask Expand grows the mask area outward, and Mask Scale Time controls how quickly masked dots shrink out or grow back during slide changes.

Timing

Timing controls the two slide phases. Current Time sends dots to the current slide, Next Time sends them to the next slide, and each Flicker Start slider chooses when that phase begins flickering. After the next phase, dots release back to the grid using the layer values.

Advanced Options

Advanced Options groups Grid, Flicker, and Blink. Grid handles layout, Flicker shapes the visual flicker rhythm, and Blink applies shared visibility timing across layers.

Grid

Grid sets columns, rows, spacing, and X/Y offset for the selected layer. Apply To All copies that layout to every active layer. Motion and force values come from the Grid Layers stack.

Flicker

Flicker controls the visual flicker style for vector slides, raster slides, and special overlays. Slide Speed controls slide pulse rate, Overlay Speed controls special overlay pulse rate, On/Off controls visible ratio, Wildness adds timing variation, and Old/New biases the outgoing and incoming flicker feel.

Blink

Blink applies shared visibility timing to the dot layers. On Period is visible time, Off Period is hidden time, Randomness varies the timing, and Probability selects how many dots participate.

Background

Image loads a PNG or JPG behind the dots. Canvas Color controls the color or color cycle inside the stage canvas and includes canvas width and height. Background Color controls the app backdrop outside the canvas and can be solid or gradient.

Presets

Add Preset saves the current grid layer stack and random limits. The X removes the selected preset unless it is the last one. Export writes all presets to JSON. Import loads a saved preset file and replaces the current list.

Save Settings

Save Settings stores timing, media, masks, special overlays, background, mouse, blink, stage, and view options. It keeps grid layer values untouched; use Presets for layer stacks.
