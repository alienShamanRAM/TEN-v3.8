Tooltips

Shortcuts

Up hides the UI. Down restores it. Left and Right perform slide flicker. Space attracts dots to the current slide. Esc stops flicker.

Overview

SLIDES displays SVG and Image slides on a fixed canvas and animates active dot-grid layers between grid positions and slide target positions. The canvas, background image layer, slides, masks, and grid coordinates share the same center-based stage coordinate system.

Header Controls

Left arrow or keyboard Left goes to the previous slide (SVG or Image). Right arrow or keyboard Right goes to the next slide. The center hold button or Space attracts dots to the current slide while held. Keyboard Up hides the UI, and keyboard Down restores it. The fast-forward button runs automatic slide changes every four seconds.

Layers

Layers controls the physics and appearance of active dot layers. Use the eye button to hide a layer, add layers above or below the selection, rename layers, delete layers, and reorder them. Target Type chooses whether that layer follows SVG fills, paths, anchors, or combined target data.

Grid

Grid defines columns, rows, spacing, and X/Y offset for each layer. Select a layer, adjust its grid, or use Apply To All to copy the selected layout to every layer.

Blink

Blink applies shared visibility timing to the dot layers. On Period is visible time, Off Period is hidden time, Randomness varies the timing, and Probability selects how many dots participate.

SVG Slides

SVG Slides loads SVG artwork as dot targets. Scale and offset position the SVG on the canvas. SVG Mask hides dots whose grid homes sit inside or near painted SVG artwork. Mask Scale Time controls how masked dots scale down or back up during slide changes.

Media Slides

Media Slides loads PNG or JPG images, or a video sequence, as rectangular dot targets. Each media file becomes a rectangle with corner anchors, edge paths, and a filled interior. Scale and offset position the media on the canvas. Media Mask hides dots inside the media rectangle. Media slides and SVG slides share the same transition sequence and flicker timing.

Flicker

Flicker controls transition timing for both SVG and Image slides. Hold Current pulls dots into the current slide, Move To Next sends dots toward the next slide, Return To Grid settles dots back into the grid, Duration controls flicker length, Old/New biases the swap, Speed controls flicker rate, On/Off controls visible ratio, and Wildness adds timing variation.

Background

Image loads a PNG or JPG behind the dots. Canvas Color controls the color or color cycle inside the stage canvas and includes canvas width and height. Background Color controls the app backdrop outside the canvas and can be solid or gradient.

Presets

Add Preset saves the current settings and adds them to the preset list. The X removes the selected preset unless it is the last one. Export writes all presets to JSON. Import loads a saved preset file and replaces the current list.

View

View Scale changes preview zoom only. It does not resize the canvas or move grid/slide coordinates. Frame Cap limits animation FPS. Info Overlay shows live canvas and runtime data at the bottom of the screen.
